Redefining the way we search for information

Design brief

Academic project

Planning | Strategy | Vision

My aim was to create a next generation search engine which acts as an umbrella platform for user in every aspect of search. Project further explores the possibility of creating a next mode of search

I partnered with three of project members and a senior design student in this project. We conducted user interviews, rapid brainstorming sessions and redefined the way we search for any information by the end of our research

Prototyping Design Concepts

Aerial Search

A novel concept inspired by short range radio communication. The UserA can set a question in his/her account and then he can simply press a button in order to send a short term message to local circle. The users (say B) inside the range can get an instant notification. The corresponding UserB can help User A in his/her query in return UserB gets reward points which can be used for shopping, recharge etc. The users can meet each other for further clarification through instant chat considering the urgency and proximity of location.

User Rated Search

In this concept friends, expert on the subject can rate certain results. Consequently, these ratings on specified search results can enhance the authenticity and grow user satisfaction to familiarity in terms of use.

Enhanced Location Based Search Modifier

The user’s location can be detected through his/her smart phone on permission. There by search results can be modified/updated precisely according to his/her needs. The user just has to check the job he/she is doing at that moment from the option menu. The user can view the most frequent check-in in a locality e.g. a social event, a party, a road side music show etc.

User Guide

A novice user can be assisted with screen pop-ups and informations
